Mah-Jungle!

Mah-Jungle! is an animal themed Mahjong puzzle game, featuring vibrant environments, dynamic board layouts, and a fun, atmospheric soundtrack. Each world features its own levels and "Special Moves", obtained by matching a specific animal. Designed for mobile platforms, and currently in development. (Gallery, visuals and design subject to change.)

Game Development & QA Insights

This is a solo project where I am handling both development and quality assurance. The focus is on creating a fun, charming and replayable game, with an UI optimized for mobile devices. Primarily made for a younger and casual target audience in mind, but features arcade-type modes which give the more experienced gamer a challenge as well.

Mode Select

Main menu, showcasing currently available modes. Classic mode with level progression, Time attack scrolls through levels and extend time for each tile pair matched / completed level.

Level Selection

Basic functional level selection screen. Currently 2 worlds implemented with 7 levels each. Features a calm mode toggle to disable time limit.

Tongue Animation

Fun little animation - the chameleon mascots shoot out their tongues and eat the tiles as they are matched.

High Score System

High scores are recorded for both modes. Classic mode for individual levels with "3-Star system", Time attack records total amount of levels cleared and matches made.

Settings Menu

Settings Menu - currently allows for audio level adjustments. When the game is paused, the board is hidden.
